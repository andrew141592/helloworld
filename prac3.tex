\section{Практическая работа №3 Построение графиков функций}
    Дан график $х^3-7х+6=0$.Интервал $[-3,3;2,8]$

Первое уравнение касательной данного графика в точке $x_0=0,11$ будет иметь вид $-6,9637*x+5,997338$.

Второе уравнения касательной этого графика в точке $ x_0=0,86 $ будет иметь вид $-4,7812*x+4,727888$

\begin{tikzpicture}
\begin{scope}[scale=0.3,xscale=8,yscale=1]

\draw[thin, ->] (-4,0) -- (3.5,0) node[right] {$X$};
\draw[thin, ->] (0,-15) -- (0,15) node[left] {$Y$};
\draw[thin, densely dashed] (0.11,0) -- (0.11,5.231331);
\draw[thin] (0.86,0) -- (0.86,0.616056);

\foreach \x\xtext in
{-3/-3,-2/-2,2/2,-1/-1,1/1,3/3} %
\draw (\x,0.1) -- (\x,-0.1) node[below] {$\xtext$};


\draw[domain=-3.3:2.8, smooth, green] plot ({\x},{((\x)*(\x)*(\x))-(7*(\x))+6});
\draw[domain=-1:2, smooth, help lines, red] plot ({\x},{(-6.9637*(\x))+5.997338});
\draw[domain=-1:2, smooth, help lines, blue] plot ({\x},{(-4.7812*(\x))+4.727888});

\end{scope};
\end{tikzpicture}
