\section{Практическая работа №3 Построение графиков функций}
    Дан график $х^3-7х+6=0$.Интервал $[-3,3;2,8]$ 

Первое уравнение касательной данного графика в точке $x_0=0,11$ будет иметь вид $-6,9637*x+5,997338$.

Второе уравнения касательной этого графика в точке $ x_0=0,95 $ будет иметь вид $-4,2925*x+4,28525$

\begin{tikzpicture}
\begin{scope}[scale=0.8]

\draw[thin, ->] (-10,0) -- (10,0) node[right] {$X$};
\draw[thin, ->] (0,-10) -- (0,10) node[left] {$Y$};

\foreach \x\xtext in
{-3/-3,-2/-2,2/2,-1/-1,1/1,3/3} %
\draw (\x,0.1) -- (\x,-0.1) node[below] {$\xtext$};


\draw[domain=-3.3:2.8, smooth, green] plot ({\x},{((\x)*(\x)*(\x))-(7*(\x))+6});
\draw[domain=-1:3, smooth, help lines, red] plot ({\x},{(-6.9637*(\x))+5.997338});
\draw[domain=-1:2, smooth, help lines, blue] plot ({\x},{(-4.2925*(\x))+4.28525});

\end{scope};
\end{tikzpicture}
